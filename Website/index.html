• We have a variable, defined as $string, which is a string 200 characters long, and we want to display only 40 characters in a snippet. What approach would you take to achieve this?

    I decided to create a proper function to make sure that both length and string can be changed... it also allows the end user to define whether they would like the cutoff point to be at the end of a word or igonre words completely.

    " $length = 40;
    function truncate($string, $length, $stopanywhere=false) {
    //truncates a string to a certain char length, stopping on a word if not specified otherwise.
    if (strlen($string) > $length) {
    //limit hit!
    $string = substr($string,0,($length -3));
    if ($stopanywhere) {
    //stop anywhere
    $string .= '...';
    } else{
    //stop on a word.
    $string = substr($string,0,strrpos($string,' ')).'...';
    }
    }
    return $string;
    }"

  • We know that $string is a url, what we need to know is the domain name used in the url. What approach would you take to achieve this?

    There is the approach of using the 'parse_url()' built in function.
    $parse = parse_url($string); $string = $parse['host'];
    However, this does not guarantee that the "www" will be removed, therefore i decided to create another function that would be much better suited to eliminate any chance that anything other than the domain would be included in the result (this was a pain to make because PHP struggles with seperating the 'host' and 'domain'). This function works with both ".com" and ".co.uk" handles from my testing.

    function getDomain($string){

    $pieces = parse_url($string);
    $domain = isset($pieces['host']) ? $pieces['host'] : '';

    if(preg_match('/(?P<domain>[a-z0-9][a-z0-9\-]{1,63}\.[a-z\.]{2,6})$/i, $domain, $regs)){
    return $regs['domain'];
    }
    return FALSE;
    }
